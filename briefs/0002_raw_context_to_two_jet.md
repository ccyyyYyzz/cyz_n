## Adopted input theorem

At a regular physical character \(x\), suppose raw experiments have already
determined

\[
U_x:W\to J_x^2,\qquad B_x:W\to\mathbb R,
\]

with

\[
0\to K_x\to J_x^2\xrightarrow{\pi_x}J_x^1\to0.
\]

For

\[
Z_x=\ker(\pi_xU_x),\qquad
H_x=U_x|_{Z_x},\qquad
\beta_x=B_x|_{Z_x},
\]

the compatible principal fiber is exactly

\[
\mathcal Q_x(D)=\{q\in K_x^*:qH_x=\beta_x\}.
\]

This finite linear theorem has been independently checked. In visible
dimension four, \(\beta_x=0\) and \(\operatorname{rank}H_x=9\) identify one
conformal scalar principal ray; rank ten fixes its scale.

## Exact bottleneck

The symbols \(J_x^2\), \(j_x^2\), \(\mathfrak m_x\) and source-fiber
evaluation cannot simply be supplied as geometric oracles. The apparatus
starts with:

- opaque repeatable source and gate tokens;
- finite source superposition;
- finite contextual gate words;
- output gates and linear readouts;
- interval-valued response tensors;
- a separately bounded response-realization class \(\mathcal C_R(n_J)\);
- a separately bounded event-algebra class \(\mathcal C_X(B_X)\).

We need the weakest auditable conditions under which those raw operations
construct a response algebra character \(x\), its filtered quotient through
order two, and the source fiber appearing in the adopted theorem.
