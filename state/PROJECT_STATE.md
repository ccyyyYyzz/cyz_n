# Project state

Updated: 2026-07-22

## Central objective

Construct an operational, non-circular definition of visible spacetime
dimension, recover a response-derived characteristic cone and its signature,
and only then ask which dynamics make a \(3+1\) visible structure stable and
universal.

## Non-negotiable distinctions

The following are different quantities:

1. laboratory control-parameter dimension;
2. response-Hankel or realization order;
3. dimension of an operational event quotient;
4. visible spacetime dimension at a declared scale/intervention algebra;
5. ambient or ontological dimension.

Total visible dimension four also does not imply signature \(3+1\).

## Established boundary

For any registered experiment closed under adding a completely blind factor,
ambient dimension is not identifiable: a model and its product with an
arbitrarily high-dimensional hidden system give identical registered
responses.  At finite resolution, sufficiently weak extra directions also
cannot be excluded without a response-gap or model-class promise.

The estimand must therefore be visible structure relative to a declared
intervention algebra, bandwidth and completeness promise.

Let \(\mathcal A\) be the complete declared commutative response algebra and

\[
X_{\mathbb R}(\mathcal A)
=\operatorname{Hom}_{\mathbb R\text{-alg}}(\mathcal A,\mathbb R).
\]

At a real character \(x\), define

\[
T_x^{*,\mathrm{vis}}
=\mathfrak m_x/\mathfrak m_x^2,
\qquad \mathfrak m_x=\ker x.
\]

For a finite-type algebra with a regular local ring at \(x\), this is the
cotangent space of a local smooth character manifold.  Physical interpretation
also requires state completeness.  A finite observed subalgebra cannot prove
that it is the complete response algebra; adjoining unobserved generators adds
arbitrarily many invisible cotangent directions while preserving all observed
algebraic data.

## Independent class bounds

All certification statements quantify over three separately declared classes:

- response realization/context class \(\mathcal C_R(n_J)\);
- event-response algebra class \(\mathcal C_X(B_X)\);
- local dynamics/generator class \(\mathcal C_L(B_L)\).

No one of these bounds is allowed to stand in for another.

## Validation model

A scalar field on \(\mathbb M^{3,1}\times S_R^1\) currently supplies the main
countermodel:

- uniform source or detector profiles give exact blindness to all nonzero KK
  modes;
- below threshold and at finite separation, extra modes are evanescent and
  low-energy corrections can be absorbed into a finite four-dimensional EFT;
- crossing KK thresholds exposes a mass-squared ladder and profile harmonics;
- all linear data remain exactly reproducible by a four-dimensional internal
  field tower, so visible compact structure is not unique ambient ontology.

## Three-state semantics

Every finite procedure must return exactly one of:

- certified within a declared class;
- declared class excluded;
- inconclusive.

It may never convert failure to see an extra direction into proof that no such
direction exists.
