# Decision 0003 — adopt the contextual Loewy boundary

Date: 2026-07-22

## Decision

Response 0002 is adopted as the raw-context front end, subject to the scope
corrections below.

The raw apparatus first reconstructs a saturated typed substitution
congruence. Its scalar quotient

\[
\widehat{\mathcal A}_D
\]

and a multiplicative physical-handle row \(\widehat x\) canonically determine

\[
\widehat J^2
=\widehat{\mathcal A}_D/
\widehat{\mathfrak m}_{\widehat x}^{\,3},
\qquad
\widehat J^1
=\widehat{\mathcal A}_D/
\widehat{\mathfrak m}_{\widehat x}^{\,2},
\qquad
\widehat K
=\widehat{\mathfrak m}_{\widehat x}^{\,2}/
\widehat{\mathfrak m}_{\widehat x}^{\,3}.
\]

This is the strongest two-jet-shaped object that is functorially determined
by the complete registered contexts.

For every consistent event model \(M\), algebraic relations are contextually
null, so there is a canonical filtered surjection

\[
\pi_{M,x}^{(2)}:J_{M,x}^2\longrightarrow\widehat J^2.
\]

It is an isomorphism exactly when its kernel vanishes, equivalently when

\[
\dim J_{M,x}^2=\dim\widehat J^2.
\]

Therefore a class-relative upper bound matching the raw quotient dimension
gives a sharp dimension-squeeze certificate. More generally, a bounded event
class may be checked for a pair of equal contextual signatures with unequal
algebraically computed jets. The absence of such a two-jet alias witness is
the exact class-relative criterion.

The response-realization bound \(n_J\) closes the contextual congruence only.
It does not provide the event-jet upper bound.

## Scope correction 1 — the raw algebra is an Artin truncation

A finite represented commutative algebra is Artinian. Its spectrum has Krull
dimension zero. If

\[
\dim\widehat{\mathfrak m}/\widehat{\mathfrak m}^{\,2}>0,
\]

the represented local algebra itself is nonregular and nonreduced; it is a fat
point, not a positive-dimensional event neighborhood.

Thus \(\widehat J^2\) is adopted as a generated visible second-order event
object. Calling it the two-jet of a regular event branch additionally requires
one of:

1. an audited regular-germ lift in \(\mathcal C_X(B_X)\); or
2. reconstruction of a bounded finite presentation

   \[
   \mathcal A=\mathbb R[z_1,\ldots,z_g]/I
   \]

   with relation completeness, a physical real branch and a
   Jacobian--Krull regularity certificate.

The smooth plane and a germ with its first relation in degree three have the
same two-jet, so this extra gate is unavoidable.

## Scope correction 2 — centered sources remove the conformal calibration

Response 0002 correctly proves

\[
\operatorname{Hom}_{\mathcal A}(S,\mathbb R_x)
\simeq(S/\mathfrak m_xS)^*.
\]

Its count of \(r_Z\) source-fiber calibrations is minimal only when
\(\beta_x\) is otherwise unrestricted on the prepared source fiber.

For the adopted conformal experiment, the same centered response algebra acts
on a registered source anchor. Every prepared source contains at least one
centered factor and therefore lies in \(\mathfrak m_xS\). Hence

\[
B_x|_W=0,
\qquad
\beta_x=0
\]

follows from the source-module structure with no point-amplitude calibration.
An independent nonzero source-fiber calibration is needed only for a general
inhomogeneous design or for absolute principal scale.

## Scope correction 3 — scalar anchoring and the general alias test

Response recycling is a clean sufficient operation that places Green outputs
in the scalar event algebra. An explicitly registered, jointly separating
\(\mathcal A\)-linear anchoring map is equally valid. Which is minimal depends
on the typed hardware model; recycling itself is not declared universally
minimal.

The general alias criterion assumes that candidate jet algebras have already
been identified with a common filtered object. In implementation this
identification must be supplied by registered generator classes or introduced
as an isomorphism variable in the bounded feasibility problem. The
dimension-squeeze route avoids that extra comparison whenever it applies.

The finite-noise theorem in response 0002 is retained as a certification
architecture. Convergence of the stated jet and source tubes still needs a
separate compactness and conditioning proof for the chosen bounded class.

## Composition with the adopted principal theorem

After scalar anchoring and either data-faithful or class-pinned jet recovery,
the result supplies

\[
U_x:W\to J_x^2.
\]

For centered sources \(B_x=0\). The already adopted theorem then gives

\[
Z_x=\ker(\pi_xU_x),\qquad
H_x=U_x|_{Z_x},
\]

\[
\mathcal Q_x(D)=\ker H_x^*.
\]

On a regular four-dimensional branch,

\[
\operatorname{rank}H_x=9
\]

identifies one nonzero conformal principal line. Constant-rank bundle gluing
and a separate signature gap promote these pointwise lines to a Lorentzian
conformal field; an ordered component-pure response germ is then required for
time orientation.
